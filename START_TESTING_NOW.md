## ✅ Procédure Simple

### Pour Chaque Modèle:

1. **Ouvrez Windsurf** (si pas déjà ouvert)
2. **Sélectionnez le modèle** dans le menu déroulant en haut du chat Cascade
3. **Collez le prompt** dans le chat
4. **Appuyez sur Entrée**
5. **Attendez la réponse** (5-30 secondes)
6. **Copiez la réponse complète**
7. **Collez-la** dans le fichier `WINDSURF_MANUAL_TEST_RESULTS_TEMPLATE.md`
