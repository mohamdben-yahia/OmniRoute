## 🔍 DÉCOUVERTES TECHNIQUES CLÉS

### Protocole
```
Type:           gRPC/Connect Protocol Version 1
Encodage:       Protobuf binaire
Transport:      HTTP/1.1 avec SSE streaming
Content-Type:   application/proto
```

### Authentification
```
CSRF Token:     x-codeium-csrf-token (UUID v4)
Session Token:  JWT dans le payload Protobuf
Format CSRF:    a5d004fc-a32d-49ab-ab4d-3d27db4167f9
Format Session: eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```

### Endpoints
```
1. StartCascade              - Initialise une session
2. SendUserCascadeMessage    - Envoie un message utilisateur
3. AssignModel               - Assigne un modèle à la session
```

### Configuration
```
Port:           Dynamique (50000-60000)
Host Header:    v.localhost:PORT
Base URL:       http://127.0.0.1:PORT
Détection:      Automatique via windsurf_server_detector.py
```
