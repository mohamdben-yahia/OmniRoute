───────────────────────────────────────────────────────────────────────────────
OVERVIEW
───────────────────────────────────────────────────────────────────────────────

The Windsurf Authentication Toolkit is a complete solution for capturing and
using authentication tokens from Windsurf IDE sessions via Chrome DevTools
Protocol (CDP). The toolkit enables automated testing and integration of
Windsurf's Language Server into external applications like OmniRoute.
